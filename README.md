# 三农政策的乡村振兴职业画像测试

一个纯静态的职业画像测试页面，围绕政策理解、产业协作和基层沟通生成乡村振兴相关画像。

## 使用

https://sannong-xg.netlify.app/

## 本地运行

```bash
npm run start
```

然后访问 `http://localhost:8080`。

## 检查

```bash
npm run check
```

当前检查会执行 `node --check app.js`，用于确认核心脚本没有语法错误。

## 主要流程

首页 -> 答题页 -> 测试结果 -> 画像详情 -> 情景模拟 -> 情景详情

未完成全部题目时，结果、画像详情和情景模拟页面会显示继续答题提示。
